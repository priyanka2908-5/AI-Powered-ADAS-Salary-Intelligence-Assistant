# 🚘 ADAS Salary Intelligence Assistant

### Transforming Raw Automotive Career Data into Actionable Salary Intelligence

ADAS Salary Intelligence Assistant is an AI-powered analytics platform that combines data intelligence and Generative AI to help users explore salary trends, skill demands, and career opportunities in the Advanced Driver Assistance Systems (ADAS) domain.

Built using Python, Streamlit, Pandas, and Google Gemini, the system enables users to interact with ADAS salary datasets through natural language conversations and receive data-driven insights instantly.

---

## 🌟 Problem Statement

Salary information in the ADAS industry is often scattered across multiple sources, making it difficult for students, professionals, and recruiters to understand market trends and compensation benchmarks.

This project addresses that challenge by consolidating salary data into an intelligent assistant capable of answering career and compensation-related questions.
